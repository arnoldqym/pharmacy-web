# Pharmacy Management System

A robust, full-stack web application designed to streamline pharmacy operations, including inventory management, prescription tracking, and point-of-sale (POS) functionality.

- **Supplier Management:** Track medicine suppliers and purchase orders.
- **User Authentication:** Secure login with role-based access control (Admin, Pharmacist, Staff).
- **Reports & Analytics:** Visual insights into daily sales, low-stock alerts, and expired products.
